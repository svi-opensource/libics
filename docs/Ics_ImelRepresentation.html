Ics_ImelRepresentation structure

DataType

Numeric representation for the imels.

type: Ics_DataType

SigBits

Number of significant bits. The default value (0), means all bits are significant.

type: size_t

Origin

Offset for imel values measured in Unit.

type: double

Scale

Scaling for imel values measured in Unit.

type: double

Unit

Units which which Origin and Scale are measured.

type: char [ICS_STRLEN_TOKEN]
